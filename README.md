Este projeto foi desenvolvido como um material interativo e dinâmico para explicar como as empresas manipulam a percepção pública com falsas promessas de sustentabilidade, mascarando seus reais impactos ambientais.

🔗 **[Clique aqui para ver a apresentação ao vivo!](https://guibazon.github.io/greenwashing/)**

## 🎯 Sobre o Projeto

Cansados dos slides tradicionais e monótonos, decidimos inovar. Esta apresentação foi construída inteiramente do zero usando tecnologias web nativas para criar uma experiência fluida, interativa e que funciona tanto no computador quanto no celular (via projetor ou na palma da mão da audiência).

### ✨ Destaques Técnicos:
* **100% Vanilla:** Construído apenas com HTML5, CSS3 e JavaScript puro. Sem bibliotecas ou frameworks pesados.
* **Design Responsivo:** O layout se adapta automaticamente a telas grandes e smartphones.
* **Navegação Intuitiva:** Barra de progresso inteligente, botões laterais invisíveis para toque e suporte a navegação por teclado (setas e espaço).
* **Dinâmicas Embutidas:** Slides formatados no estilo "Pergunta/Resposta" para interação em tempo real com o público.

## 🚀 Como executar localmente

Se você quiser baixar e rodar essa apresentação no seu próprio computador, é muito simples:

1. Faça o clone deste repositório ou baixe o arquivo `.zip`.
2. Extraia os arquivos em uma pasta.
3. Dê um duplo clique no arquivo `index.html`.
4. Ele abrirá automaticamente no seu navegador padrão (Chrome, Edge, Firefox, etc). Não é necessário internet para apresentar!
